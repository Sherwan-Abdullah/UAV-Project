The script files are to record the metrics of:

- all_data_extract.py extracts all the data (Radio, Delay, and Throughput) from the logfiles into comma separated value text files



======================================================================================================
======================================================================================================


if you want to extract data from one of the logfiles use below scripts:

- RAN_data_extract.py extracts and arranges Radio Access Network parameters into a Comma-Separated Value text file and ready it to be visualized


- Delay_data_extract.py extracts and arranges the RTT data into a Comma-Separated Value text file.


- throughput_data_extract.py extracts and arranges the Throughput data into a Comma-Separated Value text file.
